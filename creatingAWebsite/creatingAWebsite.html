Creating Your Own Webpage On Github

  1. What You Will Need

    • Index.html → This file contains your html code.
      • To get you started, here is an example of a basic HTML structure and explanations on the different elements that make up that structure.
    • Readme.md → This file contains useful information about your project.
      • Project Name » Specific and descriptive. It can also contain brief information about what your project does.
      • Description » What is the project for? What are its features?
      • Usage » Examples on how to use the project and the expected outcomes.
      • Support » Tells visitors where to go for help regarding the project (email, Twitter, etc.)
      • Roadmap » Information about any future fixes, improvements, and/or changes that might be planned for the project in the future.
      • License Information ... For open source projects, a description on how they're licensed.
      • Here is an example of a well formatted Readme file.

  2. Navigate To Github Website

    • Create profile » Sign Up and follow the procedures to create new user profile.

  3. Create New Repository

    • Choose a name for your new repository
    • Specify whether you want your repository to be Public or Private.


  4. Add Files To Your New Repository

    • Create or Upload files
    • "Commit" your changes to proceed.
